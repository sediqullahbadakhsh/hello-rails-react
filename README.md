# Hello-react-rails

> this is a greeting app build with react-redux as front-end and rails and postgres as back -end

## Built With

- _**React**_
- _**Redux**_
- _**Ruby**_
- _**Rails**_
- _**PostgreSQL**_

## Getting Started

### Clone this repository

```bash
$ git clone https://github.com/sediqullahbadakhsh/hello-react-rails.git
$ cd hello-react-rails
```

### Install gems

```bash
$ bundle install
```

### Install dependences

```bash
$ npm install
```

### Setup database

```bash
$ rails db:create
```

### Start server

```bash
$ rails s
```

Open http://localhost:3000/ in your browser

### Run tests

```bash
$ bundle exec rspec
```
